   3.交互:
 * javascript 与后端交互
   * __HTML赋值,使用JS获取__  适合传递简单数据，也非常适合多个简单数据与 Element 绑定关系。
   * _JS赋值_  将数据填充到 <script> 的 JavaScript 变量声明中。需要最快速度传递数据给 JS 并十分确定此数据稳定时，使用此方式。数据格式复杂的建议使用script填充JSON 或AJAX获取JSON 方法。
